Ruby Docs vs. JavaScript Docs

What about them? Why are they different?

(ノಠ益ಠ)ノ

10_3_2015

Let me start off by thanking all of the contributers for both of these documents. Without these sources there would be newby programmers running around everywhere ripping their hair out.

Ruby Docs, http://ruby-doc.org/ and JavaScript Docs, https://developer.mozilla.org/en-US/docs/Web/JavaScript are both free recourses that are hosted at the benefit of every programmer everywhere. I can't stress that enough, they are free, and if you ever want to give back don't hesitate to do so. So, what are the differences? Is one better than the other?

If you haven't notices already. Ruby Docs only has information on Ruby not JavaScript, and vice versa with the JavaScript Docs. Ruby Docs is maintained by contributers and is hosted and controlled by James Britt. JavaScript Docs is maintained by contributers and hosted by the Mozilla Developer Network. Thanks guys!.

Personally, I find the Ruby Docs to be a little more satisfying to use. The information is clear and the examples are short and sweet. I also think that Ruby Docs does a good job of speaking to you on a human level; they really try to cut out the technical jargon that isn't important. JavaScript Docs, sometimes, has too much technical language in the examples that can make it hard for a new programmer to understand. Other than that, JavaScript Docs looks just as good as the Ruby Docs (aside from the syntax, but that's just JavaScript for you...go Ruby!).

To any new user I would recommend going straight to the search bar at the top left corner of both sites to find what you are looking for. Want information on an array? Just type “array”. You might notice that JavaScript Docs website is a little bit busier and it might take you a few extra clicks to get where you are going. That's because the Mozilla Developer Network had more information on it than just JavaScript while Ruby Docs just has Ruby.

What's that? You don't like either of them? Well, here are some alternatives.
Ruby: http://docs.ruby-lang.org/
JavaScript: https://devdocs.io/javascript/
Seriously look around https://devdocs.io/. It has all programming languages!
